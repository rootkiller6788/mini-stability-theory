# LaSalle Invariance Principle

## Theorem (LaSalle 1960)
For x'=f(x) with V continuously differentiable, if:
1. Omega is compact and positively invariant
2. V'(x) <= 0 for all x in Omega
Then every trajectory starting in Omega approaches the largest
invariant set M in E = {x in Omega: V'(x)=0}.

## Significance
Extends Lyapunov's direct method to cases where V' is only
negative semi-definite. Does not require V' < 0 everywhere.
Particularly useful for mechanical systems with damping.

## Barbashin-Krasovskii Theorem
Special case: if V is radially unbounded and V' < 0 everywhere
except at equilibrium, then the equilibrium is GAS.
